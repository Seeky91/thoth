Invoke the maintainability skill in **audit** mode. Arguments: $ARGUMENTS

If `$ARGUMENTS` is empty, run an audit with autonomous zone selection (mode `audit auto`). If `$ARGUMENTS` is a path, audit that zone directly (mode `audit forcĂ©`). See "Mode : audit" in the skill's `SKILL.md`.
